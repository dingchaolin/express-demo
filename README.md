## express 学习 笔记
# 参数
- 1. req.path   请求的路径
- 2. req.query  GET  请求的查询字符串对象
- 3. req.body   POST 请求的请求体对象
- 4. res.send   express发送回应
- 5. req.params 对象  存放请求参数对象
- 6. req.param  方法  获取请求对象中某个字段的值
